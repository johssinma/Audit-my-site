# Product Launch Checklist

## 8 Weeks Before Launch

### Strategy

* [ ] Define launch goals (revenue target, number of users, awareness)
* [ ] Identify target segments for the launch
* [ ] Choose launch type (Product Hunt, email, social, paid, partner, community)
* [ ] Set the launch date and time
* [ ] Define success metrics and how to measure them

### Product

* [ ] Product is feature-complete for the launch version
* [ ] Landing page is built and optimised
* [ ] Pricing is finalised
* [ ] Onboarding journey is tested end-to-end
* [ ] Help documentation / FAQ is written

---

## 4 Weeks Before Launch

### Content

* [ ] Write the launch blog post / announcement
* [ ] Create the email sequence (teaser, launch, follow-up, close)
* [ ] Prepare social media posts for launch week (all platforms)
* [ ] Record the product demo video
* [ ] Create launch visuals / images
* [ ] Write the press release or media pitch (if applicable)

### Audience Building

* [ ] Set up a waitlist / early access page
* [ ] Start teasing on social media
* [ ] Contact 10-20 potential beta users
* [ ] Contact potential launch partners or affiliates
* [ ] Identify 5-10 influencers or creators to notify

### Technical

* [ ] Set up analytics tracking for launch pages
* [ ] Test the payment flow end-to-end
* [ ] Set up email automations
* [ ] Set up UTM parameters for all launch links
* [ ] Load test infrastructure (if applicable)

---

## 1 Week Before Launch

### Final Preparation

* [ ] Send teaser email to full list
* [ ] Schedule all social media posts
* [ ] Brief team members on the launch plan
* [ ] Prepare support responses for common questions
* [ ] Set up launch day war room (Slack channel, analytics dashboard)
* [ ] Re-test all links, forms and payment flows

### Community

* [ ] Prepare Product Hunt listing (if applicable)
* [ ] Draft launch day tweets / posts
* [ ] Prepare responses to anticipated questions
* [ ] Contact supporters for day-one engagement

---

## Launch Day

### Morning

* [ ] Send launch email to full list
* [ ] Post launch announcement on all social channels
* [ ] Submit to Product Hunt (if applicable)
* [ ] Notify partners, affiliates and influencers
* [ ] Monitor analytics dashboard

### Throughout the Day

* [ ] Reply to every comment and question within 1 hour
* [ ] Share user reactions and early results on social
* [ ] Send personal thank-you messages to first customers
* [ ] Monitor for bugs or technical issues
* [ ] Post engagement updates and milestone announcements

### Evening

* [ ] Send social proof email (early results, testimonials)
* [ ] Thank your community for their support
* [ ] Document lessons learned while they're fresh

---

## Post-Launch (Weeks 1-2)

### Follow-Up

* [ ] Send follow-up emails (social proof, FAQ, deadline)
* [ ] Personally reach out to biggest supporters
* [ ] Collect and publish testimonials
* [ ] Create a case study from the best early result
* [ ] Close the launch offer (if time-limited)

### Analysis

* [ ] Compile launch metrics vs goals
* [ ] Identify the best-performing channels
* [ ] Document what worked and what didn't
* [ ] Plan post-launch content strategy
* [ ] Set 30 and 90-day goals based on launch data

---

## Key Metrics to Track

| Metric                   | How to Measure                  |
| ------------------------ | ------------------------------- |
| Total sign-ups / sales   | Payment processor + analytics   |
| Revenue                  | Stripe / payment dashboard      |
| Conversion rate          | Visitors → sign-ups            |
| Email open rate          | Email platform                  |
| Social media impressions | Platform analytics              |
| Product Hunt ranking     | Product Hunt dashboard          |
| Customer feedback        | Support inbox + social mentions |
| Referral sign-ups        | UTM tracking                    |
