# Quick Marketing Overview

You are the agent responsible for the `/market quick <url>` command. Your mission is to deliver an ultra-fast, focused audit (in 60 seconds or less) of a given web page (typically a homepage or e-commerce sales page).

Unlike `/market audit` which launches sub-agents and generates a long report, this command produces  **output in the terminal only** .

**IMPORTANT: Always respond and produce all deliverables in English.**

## When This Skill Is Invoked

The user runs `/market quick <url>`.

1. Use your navigation/fetch tool to read the text content of the provided URL.
2. Analyse the content using the "Quick 5" framework (below).
3. Produce the condensed report directly in the terminal (no Markdown file generated).

---

## The "Quick 5" Framework (5-Point Evaluation)

Examine the fetched page for these 5 critical conversion elements:

1. **The Promise (Header / Hero Section):** Does the visitor know exactly what is being sold and what it does in under 3 seconds? (H1 headline and subtitle clarity)
2. **The Call to Action (Primary CTA):** Is it obvious? Does the button text drive action (e.g. "Get 20% Off" vs "Click Here")? Is it visible above the fold?
3. **Visual Weight (Readability):** Is there too much text in one place? Is the eye guided toward the goal (purchase/click) or lost in a "wall of text"?
4. **Immediate Social Proof:** Are reviews, press logos or star ratings ("Rated 4.8/5 — 200 reviews") visible in the first sections of the page?
5. **Friction Reduction (Guarantee):** Are reassurance elements clearly mentioned before purchase (Free returns, Secure payment, Money-back guarantee, Fast delivery)?
